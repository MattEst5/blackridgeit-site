# BlackRidge IT Website

This repository contains the static website for BlackRidge IT, a small business IT support service focused on practical technology support, system setup, backup verification, software troubleshooting, and monthly support for local businesses.

## Project Overview

This site was originally built as part of CS50x's Homepage assignment and is being developed into the live business website for BlackRidge IT.

## Pages

- `index.html` - Homepage
- `services.html` - Services overview
- `about.html` - About BlackRidge IT
- `contact.html` - Contact and support request form
- `thank-you.html` - Form submission confirmation page

## Technologies Used

- HTML
- CSS
- JavaScript
- Bootstrap 5
- Netlify Forms
